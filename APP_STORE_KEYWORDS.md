# Khutbah Companion - Keyword Strategy

Complete keyword optimization for iOS App Store and Google Play Store across all three markets.

---

## 📊 KEYWORD STRATEGY OVERVIEW

### Apple App Store (100 characters max)
- No spaces, use commas to separate
- Focus on single words over phrases
- Include digits (not spelled out)
- Singular vs plural based on search data
- No competitor names

### Google Play Store
- Keywords placed naturally in description (4000 chars)
- Repeat top keywords 5 times throughout description
- Natural placement, avoid keyword stuffing
- Searchable in: title, short description, long description

## 🎓 KEYWORD BEST PRACTICES

### DO:
✅ Focus on user intent (what problem are they solving?)
✅ Balance high-volume with low-competition keywords
✅ Use exact match keywords from competitor apps
✅ Include location-based keywords for local markets
✅ Test and iterate based on data
✅ Use keyword variations (singular/plural, with/without accents)

### DON'T:
❌ Keyword stuff (will get penalized)
❌ Use competitor brand names in metadata
❌ Ignore long-tail keywords (higher conversion)
❌ Set and forget (requires ongoing optimization)
❌ Copy-paste exact competitor keywords without analysis
❌ Use irrelevant high-volume keywords just for traffic
